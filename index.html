Учить слова

Писать иероглифы

Тренируй каждый иероглиф отдельно. Если слово состоит из двух иероглифов, например 喜欢, выбери 喜 или 欢.

Запоминание

Собери правильный порядок слов во фразе.

Добавить свою фразу

Пиши китайские слова через пробел. Например: 我 喜欢 茶

Соедини китайские слова справа с правильными переводами слева. Pinyin здесь больше не используется.

Вспомни и напиши иероглиф без контура и без готового символа.

Введи ханзи по переводу. Поле закреплено, чтобы страница не дёргалась при переходе.

Понимание текста

Добавляй китайский текст, слушай его целиком и нажимай на отдельные иероглифы, чтобы услышать звук. Встроенный учебный переводчик пока ищет знакомые слова по нашему словарю.
